## 請解釋後端與前端的差異。

- 前端負責網頁畫面的呈現。
- 後端為負責邏輯運算、處理資料、資料庫管理。

1. 對瀏覽器下指令
2. 指令傳給作業系統
3. 指令傳到網路卡
4.  網路卡將指令傳到 DNS 
5. DNS 告知網路卡 IP 位置
6. 網路卡發 request 給 server
7. server 去 database 找資料
8. 回傳 respose 到網路卡
9. 網路卡傳到作業系統
10. 作業系統傳回瀏覽器
11. 瀏覽器顯示出網頁

其中，步驟 1~3、9~11 為前端；4~8 為後端所負責。
